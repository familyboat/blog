前言

虽然并不总是正确的，但 rust 编程语言基本上和强化有关：无论你正在写的是哪种代码，rust 能比以往任何时候让你走的更远，以及在更广阔的领域自信的编程。以与底层的内存管理、数据表示以及协同打交道的“系统级”任务为例。通常，该领域的编程被认为是晦涩难懂的，且仅被少数花了必要时间学习过如何避免易犯陷阱的人所掌握。但这些富有经验的人仍需小心谨慎，唯恐编写的代码存在漏洞。rust 打破了这些障碍，一路上通过消除以往的陷阱以及提供友好的、精炼的工具集来帮助你。那些需要深入到底层控制的程序员使用 rust 的话，就不需要考虑程序崩溃或漏洞的常规风险，不需要学习变化很快的工具链的要点。这门语言还能更好，能指导你自然地编写出就速度和内存方面而言高效且可信赖的代码。

那些已经与底层打过交道的程序员可以通过 rust 提升信心。例如，rust 中的并行相对来说是低风险的操作：编译器会为你捕获经典的错误。并且你能自信的渐进式地优化你地代码，同时并不会意外地引入漏洞。

但 rust 并不局限于底层系统级编程。愉快的构建 CLI 应用、web 服务器以及其它种类的应用--随后，在本书中会找到一些简单的例子，也能完全胜任。使用 rust 能让你掌握从一个领域到另一个领域的技能；你可以通过编写 web 应用学习 rust，然后应用这些技能构建你自己的 raspberry PI。

本书包含了 rust 强化使用者的全部能力。本书的内容友好且易得，旨在帮助你提升 rust 的能力，以及作为程序员应有的自信。所以细细品尝吧，准备好学习吧，欢迎来到 rust 社区！

— Nicholas Matsakis 和 Aaron Turon
